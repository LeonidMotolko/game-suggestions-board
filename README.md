# Game Suggestions Board

Веб-приложение на FastAPI для сбора предложений по игре от пользователей. Поддерживает регистрацию с подтверждением email, роли (пользователь/администратор), загрузку файлов и административную панель со статистикой.

## Возможности

- 🔐 **Аутентификация**: email + пароль, подтверждение email, JWT токены.
- 👤 **Роли**: обычный пользователь и администратор.
- 📝 **Предложения**: создание, просмотр, изменение статуса (новое → рассмотрено → принято/отклонено).
- 📎 **Файлы**: прикрепление скриншотов/документов к предложению.
- 📊 **Админ-панель**: управление пользователями, предложениями, просмотр статистики.
- 🐳 **Docker**: полностью контейнеризированное окружение.

## Технологический стек

- **Backend**: FastAPI (Python 3.12)
- **Database**: PostgreSQL 16 + SQLAlchemy (асинхронный) + Alembic
- **Auth**: FastAPI Users + JWT + bcrypt
- **Frontend**: Jinja2 + HTMX (серверный рендеринг)
- **Storage**: Локальное хранилище с возможностью перехода на S3

## Быстрый старт

1. Клонируйте репозиторий:
   ```bash
   git clone <repo-url> game-suggestions
   cd game-suggestions
```

1. Скопируйте `.env.example` в `.env` и настройте переменные:

```
cp .env.example .env
```
1. Запустите приложение через Docker Compose:

```
docker compose up --build
```
1. Откройте в браузере: [http://localhost:8000](http://localhost:8000)
1. Первый администратор создаётся автоматически при первом запуске:

- Email: `admin@example.com`
- Пароль: `admin123`
**Сразу смените пароль в настройках!**
